git常用命令：

  git init   git初始化,目的让git可识别跟踪咱们的项目文件
  git status  查看文件状态
  git add 文件  添加文件
  git commit  -m '文字说明' 提交版本
  查看提交日志： git log --pretty=oneline

   其中：--pretty=oneline 在一行显示，清晰简洁

  时间穿越:git reset --hard 提交id
